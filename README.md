# Focinhos Felizes

Site em Next.js para a ONG Focinhos Felizes.

## Rodar localmente

```bash
pnpm install
pnpm dev
```

## Deploy

Conecte este repositório na Vercel e configure o domínio:

- `focinhosfelizes.ong.br`
- `www.focinhosfelizes.ong.br`

## Próximas integrações

- Supabase Auth para login administrativo.
- Supabase Database para animais, blog, leads e links de checkout.
- Supabase Storage para imagens.
- Asaas via API segura no backend.
- IA via API Route/Edge Function, sem expor chave no navegador.
